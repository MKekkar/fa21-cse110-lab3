Click to Expand

This project is very important and highly classified. Our proposal is to create a new search engine that orders results from the lowest to highest accuracy so that users can find new information they did not expect or know about.

The results which partially match their search query should be bolded, while those that do not match at all should be italicized. If any results are a direct match of the search, then they should be both bolded and italicized.
